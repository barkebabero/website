2. Finalidad

Tratamos únicamente los datos necesarios para:

  • Gestionar pedidos enviados mediante WhatsApp.
  • Atender consultas realizadas desde la web.
  • Gestionar reservas telefónicas cuando el usuario decide llamar.
  • Cumplir las obligaciones legales aplicables.

3. Datos que recopilamos

  • Nombre.
  • Número de teléfono.
  • Dirección de entrega.
  • Notas del pedido.
  • Información introducida voluntariamente por el cliente.

No solicitamos datos innecesarios.

4. Promoción de cumpleaños

Si desea utilizar la promoción de cumpleaños, será necesario presentar un documento oficial de identidad (DNI, NIE o Pasaporte) al personal del restaurante.

La verificación será únicamente visual.

No realizamos copias.

No escaneamos documentos.

No almacenamos el número del documento únicamente para esta promoción.

5. Base jurídica

El tratamiento de los datos personales se basa en:

  • El consentimiento del usuario.
  • La ejecución del pedido solicitado.
  • El cumplimiento de obligaciones legales.

6. Conservación de datos

Los datos se conservarán únicamente durante el tiempo necesario para gestionar el pedido o cumplir las obligaciones legales.

7. Destinatarios

No vendemos datos personales.

Los datos únicamente podrán ser comunicados cuando exista obligación legal o sea necesario para la prestación del servicio.

8. Derechos del usuario

Puede ejercer los siguientes derechos:

  • Acceso.
  • Rectificación.
  • Supresión.
  • Limitación.
  • Oposición.
  • Portabilidad.

Puede contactar con nosotros utilizando nuestro teléfono o visitando el restaurante.

9. Seguridad

Aplicamos medidas técnicas y organizativas razonables para proteger la información tratada.

10. Cambios

Esta política podrá actualizarse cuando sea necesario para adaptarse a cambios legales o del funcionamiento del sitio web.
